JS到底是什么?

  • JavaScript 是脚本语言
  • JavaScript 是一种轻量级的编程语言。
  • JavaScript 是可插入 HTML 页面的编程代码。
  • JavaScript 插入 HTML 页面后，可由所有的现代浏览器执行。
  • JavaScript 很容易学习

4.3 布尔值(boolean)

JS布尔值的定义：

  • 在python中布尔值是首字母大写的
    • True / False
  • 但是在js中布尔值是全小写的
    • true / false
  • 布尔值是false的有哪些
    • 空字符串
    • 0
    • null
    • undefined
    • NaN
