二、局部变量和全局变量

2.1 定义局部和全局变量

分为全局和局部（外部）变量：

  • 局部变量：定义在函数里面的变量（默认在自动存储期限，可以用static指定为静态）
  • 全局变量：定义在函数外面的变量

2.2 存储期限（※※※）

存储期限分为：

  • 自动存储期限：存在栈里面的数据，变量的生命周期随栈帧的入栈开始，出栈而结束
  • 静态存储期限：拥有永久的存储时间，在程序整个执行期间都存在

四、指针（※※※※）

4.1 指针的基础

几个基础知识：

  • 字节：计算机最小的寻址单位
  • 变量的地址：变量的第一个字节的地址
  • 指针：指针就是地址
  • 指针变量：存放地址的变量，有时候吧指针变量称为指针

4.1.1 通过指针访问对象

如何通过指针访问到其指向的对象：

注意事项：

int *p, q; // p是指针int*类型，q是int类型

int *p, *q; // p、q都是int*类型

4.1.2 野指针的问题

什么是野指针？

  • 指向未知区域的指针

  • 未初始化的指针

注意事项：

对野指针进行解引用行为，将导致编译器报错：如，读取访问权限异常
