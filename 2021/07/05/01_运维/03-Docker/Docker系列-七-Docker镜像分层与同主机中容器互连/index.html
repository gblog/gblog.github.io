иҝҗз»ҙв–әпјҲдёүпјүDocker

Dockerзі»еҲ—(дёғ)-Dockerй•ңеғҸеҲҶеұӮдёҺеҗҢдё»жңәдёӯе®№еҷЁдә’иҝһ

дёҖгҖҒDockerй•ңеғҸеҲҶеұӮ

й•ңеғҸеҲҶеұӮзҡ„еҘҪеӨ„пјҡ

вҖӢ еӨҚз”ЁгҖҒиҠӮзңҒзЈҒзӣҳз©әй—ҙпјҢзӣёеҗҢзҡ„еҶ…е®№еҸӘйңҖеҠ иҪҪдёҖд»ҪеҲ°еҶ…еӯҳ

вҖӢ дҝ®ж”№dockerfileеҗҺпјҢйҮҚж–°жһ„е»әж—¶еҸҜд»Ҙз”Ёзј“еӯҳпјҢйҖҹеәҰеҝ«

1 жҹҘзңӢdockerй•ңеғҸеҲҶеұӮ

йҖҡиҝҮеҜје…Ҙй•ңеғҸеҸҜд»ҘжҹҘзңӢеҲ°й•ңеғҸеҲҶеұӮ

йҖҡиҝҮжҹҘзңӢй•ңеғҸеҺҶеҸІеҸҜд»ҘжҹҘзңӢеҲ°еҲҶеұӮ

жүҖжңүзҡ„иҝҷдәӣеұӮйғҪдјҡеңЁDockerдё»жңәжң¬ең°еӯҳеӮЁеҢәеҹҹеҶ…еӯҳеӮЁпјҢеҸҜд»ҘйҖҡиҝҮд»ҘдёӢжҢҮд»ӨжқҘеҲ—еҮәпјҡ

2 йҖҡиҝҮDockerfileдјҳеҢ–еҲҶеұӮдҝЎжҒҜ

  • е°ҪйҮҸеҗҲе№¶RUNе’ҢADDжқҘеҮҸе°‘й•ңеғҸеҲҶеұӮж•°
  • ж–°еҠ зҡ„DockerfileиҜӯеҸҘеҠ еҲ°жңҖеҗҺпјҢдёҚиҰҒеҠ еҲ°еүҚйқў

дәҢгҖҒеҗҢдё»жңәдёӯе®№еҷЁдә’иҝһпјҲвҖ“linkжҳҜеҚ•еҗ‘зҡ„пјү

dockerе®ҳж–№е·ІдёҚжҺЁиҚҗдҪҝз”Ёdocker run вҖ“linkжқҘй“ҫжҺҘ2дёӘе®№еҷЁдә’зӣёйҖҡдҝЎпјҢйҡҸеҗҺзҡ„зүҲжң¬дёӯдјҡеҲ йҷӨвҖ“link

1 еҠҹиғҪд»Ӣз»Қ

docker run вҖ“linkеҸҜд»Ҙз”ЁжқҘй“ҫжҺҘ2дёӘе®№еҷЁпјҢдҪҝеҫ—жәҗе®№еҷЁпјҲиў«й“ҫжҺҘзҡ„е®№еҷЁпјүе’ҢжҺҘеҸ—е®№еҷЁпјҲдё»еҠЁеҺ»й“ҫжҺҘзҡ„е®№еҷЁпјүд№Ӣй—ҙеҸҜд»Ҙдә’зӣёйҖҡдҝЎпјҢе№¶дё”жҺҘж”¶е®№еҷЁеҸҜд»ҘиҺ·еҸ–жәҗе®№еҷЁзҡ„дёҖдәӣж•°жҚ®пјҢеҰӮжәҗе®№еҷЁзҡ„зҺҜеўғеҸҳйҮҸгҖӮдҪҝз”ЁжЎҲдҫӢеҰӮдёӢпјҡ

жәҗе®№еҷЁеҗҜеҠЁпјҡ

жҺҘеҸ—е®№еҷЁиҝһжҺҘпјҡ

иҝӣе…ҘжҺҘеҸ—е®№еҷЁжөӢиҜ•пјҢдёҚйңҖиҰҒping IPпјҢзӣҙжҺҘpingеҲ«еҗҚе°ұеҸҜд»ҘпјҢwebе’Ңsrc_dockerйғҪжҢҮеҗ‘172.16.0.2пјҲеҚ•еҗ‘пјү

жҺҘеҸ—е®№еҷЁзҡ„/etc/hostsе°Ҷжӣҙж–°

2 жЎҲдҫӢпјҡжһ„е»әzabbix-server

еҗҜеҠЁдёҖдёӘmysqlзҡ„е®№еҷЁ

еҗҜеҠЁjava-gatewayе®№еҷЁзӣ‘жҺ§javaжңҚеҠЎ

еҗҜеҠЁzabbix-mysqlе®№еҷЁдҪҝз”ЁlinkиҝһжҺҘmysqlдёҺjava-gateway

еҗҜеҠЁzabbix webжҳҫзӨәпјҢдҪҝз”ЁlinkиҝһжҺҘzabbix-mysqlдёҺmysql

zabbixзҡ„й»ҳи®Өз«ҜеҸЈе·Іжңү80ж”№дёә8080пјҢеҸҜи§Ғй…ҚзҪ®ж–Үд»¶/etc/zabbix/nginx.conf

зҷ»еҪ•Zabbix

ж·»еҠ иў«зӣ‘жҺ§иҠӮзӮ№-е®үиЈ…zabbix-agent

иҺ·еҸ–zabbix-agentпјҡ

uname -a жҹҘзңӢеҶ…ж ёзүҲжң¬

webйЎөйқўжҹҘзңӢzabbixзүҲжң¬

https://www.zabbix.com/download иҺ·еҸ–еҜ№еә”agentзҡ„е®үиЈ…ж–№жі•

ж·»еҠ иў«зӣ‘жҺ§иҠӮзӮ№-agentй…ҚзҪ®ж–Үд»¶дҝ®ж”№

117иЎҢпјҡServer=10.0.0.11, жіЁж„ҸйҳІзҒ«еўҷе’Ңselinuxзҡ„йҳ»жҢЎ

дёүгҖҒdockerйҮҚеҗҜеҗҺе®№еҷЁдёҚйҖҖеҮә

й»ҳи®Өжғ…еҶөдёӢпјҢsystemctl restart dockerд№ӢеҗҺпјҢе®№еҷЁе°ҶеӨ„дәҺExitedзҠ¶жҖҒ

1 ж·»еҠ е®№еҷЁеҗҜеҠЁеҸӮж•°

dockerйҮҚеҗҜеҗҺпјҢе®№еҷЁе…ҲеҒңжӯўпјҢеҶҚз«ӢеҚійҮҚж–°еҗҜеҠЁ

2 daemonй…ҚзҪ®ж–Үд»¶дҝ®ж”№пјҲдёҚжҺЁиҚҗпјү

dockerйҮҚеҗҜеҗҺпјҢе®№еҷЁдёҚдјҡеҒңжӯўпјҢдёҖзӣҙеңЁиҝҗиЎҢпјҢдёҚжҺЁиҚҗдҪҝз”ЁпјҢдёҚеҘҪжҺ§еҲ¶
